# Kaggle House Prices 売却価格予測（線形回帰）

## 概要
　本リポジトリでは、KaggleのHouse Pricesデータセットを用いて、住宅の売却価格を予測する線形回帰ベースの機械学習モデルを構築している。また、本分析は、ChatGPTを補助的に利用しながらも自力でモデル構築を行い、その後、ChatGPTの改善案を参考にしつつ、前処理、特徴量設計、モデル構築の各工程について「なぜその判断を行ったのか」を意識して改善、整理している。
## データセット
　本分析では、Kaggleが提供するHouse Pricesデータセットを使用している。また、本データセットは住宅に関する属性情報（例：建築年月、面積など）と、住宅の売却価格から構成されている。
## 目的
　本分析の目的は、House Pricesデータセットを用いて、住宅の属性情報から売却価格を予測することである。その手段として、線形回帰ベースの機械学習モデルを構築した。また本分析では、Kaggle上で高いスコアを獲得する事を重視するのではなく、各工程について「なぜその手法を選択したのか」を説明できる状態になることを重視している。
## 分析方針
　分析は以下の方針で進めた。

- まずは線形回帰モデルを用い、比較的簡単な構成で基礎となるモデルを自力で構築する
- その後、前処理や特徴量の扱いについて見直しを行い、モデルの改善を試みる
- 各工程において、実装結果だけでなく「なぜその判断を行ったのか」を言語化し、ノートブック内の注釈として整理する
## モデル
　本分析では、線形回帰（LinearRegression）を使用した。線形回帰は、各特徴量が目的変数の決定にどれぐらい影響するかを係数として学習する点に特徴があり、内部での手続きが比較的分かりやすいモデルである。学習初期段階においては、モデルの分かりやすさは重要と判断したため採用した。モデルの学習には、前処理および特徴量変換を施した訓練データを使用し、過学習を抑制するためにRidge回帰を用いて、係数が過剰に大きくなることを抑制している。
## 結果
- **Baseline（house_prices_baseline.ipynb）**
  - RMSE：0.14555（Kaggle Public Score）
- **Improved（house_prices_improved.ipynb）**
  - RMSE：0.13902（Kaggle Public Score）

## 今後の改善
本分析では、自力でのモデル構築と無理の無い応用を重視し、比較的シンプルな構成で線形回帰モデルを構築した。  
一方で、以下のような改善余地があると考えている。

- 欠損値補完を全体の中央値ではなく、地域などの属性ごとの中央値で行う方法
- 数値特徴量の標準化を行い、Ridge回帰における正則化の効果をより適切に反映させる方法
- 出現頻度の低いカテゴリをまとめることで、ダミー変数の次元数削減を行う方法
- クロスバリデーションを用いて正則化係数αの選択を行う方法

これらの手法はスコア向上に繋がる可能性があるが、本分析では「なぜその判断を行ったのか」を明確に説明できる範囲に分析を留めること、及び地に足を付けた学習を行うことを優先し、今後の課題として位置づけている。
