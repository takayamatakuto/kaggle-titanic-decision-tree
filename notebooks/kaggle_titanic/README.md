# Kaggle Titanic 生存予測（決定木）

## 概要
　本リポジトリでは、KaggleのTitanicデータセットを用いて、乗客の生存可否を予測する決定木ベースの機械学習モデルを構築している。
　また、本分析は、これまでの学習内容（実践データ分析100本ノック）の集大成として、ChatGPTを補助的に利用しながらも自力でモデル構築を行い、その後、ChatGPTの改善案を参考にしつつ、前処理、特徴量設計、モデル構築の各工程について「なぜその判断を行ったのか」を意識して改善、整理している。
## データセット
　本分析では、Kaggleが提供するTitanicデータセットを使用している。
　本データセットはタイタニック号の乗客に関する属性情報（例：客室等級、性別など）と、生存可否を示すラベルから構成されている。
## 目的
　本分析の目的は、Titanicデータセットを用いて、乗客の属性情報から生存可否を予測することである。その手段として、決定木ベースの機械学習モデルを構築した。
　また本分析では、Kaggle上で高いスコアを獲得する事を重視するのではなく、各工程について「なぜその手法を選択したのか」を説明できる状態になることを重視している。
## 分析方針
　分析は以下の方針で進めた。

・まずは決定木モデルを用い、比較的簡単な構成で基礎となるモデルを自力で構築する
・その後、前処理や特徴量の扱いについて見直しを行い、モデルの改善を試みる
・各工程において、実装結果だけでなく「なぜその判断を行ったのか」を言語化し、ノートブック内の注釈として整理する
## モデル
　本分析では、決定木分類器（Decision Tree Classifier）を使用した。決定木は、各特徴量がどのような条件で分岐し、予測に寄与しているかを可視化できる点に特徴があるため、内部での手続きが比較的分かりやすいモデルである。学習初期段階においては、モデルの分かりやすさは重要と判断したため採用した。
　モデルの学習には、前処理および特徴量変換を施した訓練データを使用し、過学習を抑制するために木の深さ（max_depth）を制限している。
## 結果
本分析では、まずベースラインモデルとして決定木分類器を用いたモデルを構築した。
- **Baseline（titanic_baseline.ipynb）**
  - Accuracy：0.74641（Kaggle Public Score）
※ 改善後モデル（titanic_improved.ipynb）については、現在作成中であり、完成後に追記する。
